# Conversor de Monedas en C

## Descripción
Aplicación desarrollada en lenguaje C que permite convertir monedas entre quetzales, dólares y euros.

## Prerrequisitos
- Tener instalado GCC
- Sistema operativo Linux o Windows
- Terminal

## Pasos para ejecución

1. Compilar el programa:
gcc conversor.c -o conversor

2. Ejecutar el programa:
./conversor
